# 旅行规划 (Travel Planning)

## 目录概述
本目录包含了北京旅行的规划材料。这是一个非代码/静态网页项目，目前由一个高度详细且可交互的单文件 HTML 行程单组成。该指南为两人规划了 2026 年 4 月 30 日至 5 月 3 日的 4 天 3 晚行程。

## 关键文件
*   **`beijing-travel-guide.html`**: 本项目的核心文件。它是一个独立的网页，包含：
    *   **行程单：** 按天划分的旅行日程（Day 1 到 Day 4）。
    *   **交互式地图：** 集成了高德地图 (AMap) API，用于可视化路线（地铁、打车、步行）和景点位置。
    *   **旅行提示与信息：** 包含门票、开放时间、预约要求（如故宫、国家博物馆）和交通费用的详细信息。
    *   **样式与逻辑：** 内嵌了用于美化 UI 的 CSS 以及用于处理交互（如切换日期、显示路线详情、显示特定景点的弹窗）的 JavaScript。

## 使用说明
由于这是一个静态 HTML 项目，不需要构建或配置服务器。

*   **查看指南：** 只需在任何现代网页浏览器（如 Chrome、Safari、Edge）中打开 `beijing-travel-guide.html` 文件。
*   **修改行程：** 直接使用文本编辑器或 IDE 编辑 `beijing-travel-guide.html` 文件。行程数据、路线、景点坐标和样式都硬编码在文件的 HTML、CSS 和 JavaScript (`<script>`) 部分中。
    *   *注意：修改地图渲染或路线逻辑可能需要了解内嵌的高德地图 (AMap) API。*
